# AI 人格画像模板

> 复制这份模板，填你自己的信息，放入你所用 AI 工具的入口文件。
> 
> **各工具的入口文件位置：**
> - Claude Code → `CLAUDE.md`（项目根目录）
> - Codex → `AGENTS.md`（项目根目录）
> - Cursor → `.cursor/rules/persona.md`
> - GitHub Copilot → `.github/copilot-instructions.md`
> - Gemini CLI → `GEMINI.md`（项目根目录）
> - 其他/不确定 → `AGENTS.md`（通用兜底）
>
> AI 读到这份文件后，会以你填的身份和偏好来工作。**同一份人格画像可以跨工具复用——内容不变，只改存放位置。**
