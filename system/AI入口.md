# AI 入口（路由层）

> 🤖 **读到即执行。** 你不是来浏览知识的，你是来解决任务的。本页是你的 GPS——按任务类型精准读取，不遍历全库。

## 读取规则

1. **先读 `[[系统/当前状态]]`** — 知道上一轮做到哪了
2. **先扫 `[[系统/AI数据层]]`** — 结构化索引，按场景关键词精准匹配目标文件
3. **按下方路由表，只读匹配当前任务的那一行** — 最多 3 个文件
4. **禁止跟 index.md 的链接** — index.md 是人看的导航，不是你的读取清单
5. **> 5KB 的文件先扫标题**（grep `^#`），确认相关再读全文
6. **80% 的任务只需要 1-2 个文件** — 多读了就是在浪费 token
